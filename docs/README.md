# Docs Index

## Structure

- `docs/architecture`
  Engineering-facing architecture diagrams and design rationale.

- `docs/specs`
  Source-of-truth product and implementation specs.

- `docs/guides`
  Setup, runbooks, and project-explainer docs.

- `docs/audits`
  Current-state assessments and problem-focused audits.

- `docs/plans`
  Execution plans and next-step checklists.

- `docs/reports`
  Completed work summaries and hardening reports.

- `docs/design`
  UI and layout design references.

- `docs/layman`
  Plain-English explanations for non-technical reviewers.

- `docs/prompts`
  Orchestrator and subagent prompt files.
